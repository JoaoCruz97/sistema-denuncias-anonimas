# Sistema de Denúncias Anônimas

Projeto acadêmico desenvolvido para a disciplina de Engenharia de Software III.

## Objetivo
Permitir que cidadãos registrem denúncias de forma anônima e recebam um comprovante em PDF.

## Funcionalidades
- Categorias de denúncia
- Localização (bairro, rua, referência)
- Campo de descrição
- Upload de evidência (simulado)
- Geração automática de protocolo
- Download automático de comprovante em PDF

## Tecnologias
HTML
CSS
JavaScript
jsPDF
